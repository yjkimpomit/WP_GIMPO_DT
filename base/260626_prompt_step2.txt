너는 UI/UX 경력 20년차 시니어 디자이너이자 퍼블리싱 전문가 기준으로 작업해줘.

첨부한 요구사항을 분석해 HTML5/CSS/JS 프로토타입을 제작하고, index.html / styles.css / app.js / README / sitemap.md / ZIP 파일로 제공해줘.

설계 기준:
- 실제 업무자가 쓰는 운영형 UI 기준
- 중앙 핵심 시각화 영역은 가리지 않음
- 좌우 패널은 동일 너비, CSS 변수로 제어
- 전체 화면은 100vh, 전체틀 스크롤 금지
- 카드마다 내부 스크롤 적용
- 카드 헤더 고정, body만 스크롤
- 스크롤바는 hover 시 6px 표시, scrollbar-gutter로 밀림 방지
- dashboard-layer를 무리하게 grid로 고정하지 말고 좌우 패널 레일 독립 배치
- 서랍 상단은 기능 메뉴, 하단은 순수 설비구조 Tree
- Selected Asset Panel은 설비정보/운전·성능 탭 구조로 구현
- 전체 패널 토글과 개별 Quick Panel 토글은 분리
- 수정 요청 시 v2, v3처럼 버전 관리해서 ZIP으로 제공

답변은 반영사항, 판단 근거, 다운로드 링크 중심으로 간결하게 해줘.
