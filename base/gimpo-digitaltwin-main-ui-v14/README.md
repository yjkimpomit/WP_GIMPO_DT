# 김포열병합 디지털트윈 플랫폼 UI v14

## 수정 반영

- 좌측 `GENi 정비/TM 통계` 카드를 상단으로 이동
- `GENi 정비/TM 통계` 내용을 2칼럼으로 재배치하여 카드 높이 축소
- 우측 `주요 운영 현황`, `Tag 연계 콘텐츠`는 콘텐츠 높이만큼만 표시
- 우측 하단 `실시간 알람`은 남은 영역을 사용하고, 넘치는 내용은 카드 내부 스크롤 처리
- 서랍 하단 설비구조에 트리 여닫기 아이콘 추가
- 설비구조는 기능 메뉴가 아닌 순수 설비명 Tree로 유지
